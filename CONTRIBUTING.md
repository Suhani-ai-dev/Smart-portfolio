# Contributing Guidelines

Thank you for your interest in contributing to this project.

This portfolio website was built to showcase my projects, skills, and experience. While this is primarily a personal project, contributions for improvements, bug fixes, and UI enhancements are welcome.

---

## How to Contribute

1. Fork this repository

2. Clone your fork

```bash
git clone https://github.com/revatikadam0607/portfolio.git
```

3. Create a new branch

```bash
git checkout -b feature-name
```

4. Make your changes

5. Commit your changes

```bash
git commit -m "Added feature"
```

6. Push changes

```bash
git push origin feature-name
```

7. Open a Pull Request

---

## Contribution Ideas

You can contribute by:

- Improving UI/UX
- Fixing bugs
- Optimizing responsiveness
- Improving animations
- Enhancing accessibility
- Improving documentation

---

## Code Style

- Keep code clean and readable
- Use meaningful variable names
- Maintain consistent formatting
- Avoid unnecessary dependencies

---

## Reporting Issues

If you find bugs or issues:

1. Open an issue
2. Describe the problem clearly
3. Add screenshots if needed
